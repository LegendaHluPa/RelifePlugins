siege - Основная команда
siege end - Закончить войну
siege start - Начать войну
siege list - Список городов участвующих в войне
siege list add {Названи города} - Добавить город в список 
siege list remove {Названи города} - Убрать город из списка
siege list clear - Очистить список городов
siegealive {Названи города} - Узнать сколько в городе осталось живых во время войны
siegedeadth allunfreeze - Дать возможность двигатся всем игрокам
siegedeadth freeze {Ник игрока} - Зафризить игрока
siegedeadth unfreeze {Ник игрока} - Убрать фриз игроку
